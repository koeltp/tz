# TP投资
TP投资是一款专注于投资美股和期权的智能投顾应用，旨在帮助用户实现财富增值和风险管理。通过先进的算法和数据分析，TP投资为用户提供个性化的投资组合建议和实时市场监控，助力用户在复杂的金融市场中做出明智的投资决策。

## 主要功能
- **个性化投资组合**：根据用户的风险偏好和投资目标，TP投资生成量身定制的投资组合，涵盖多种资产类别。
- **实时市场监控**：提供实时的市场数据和分析，帮助用户把握投资机会。
- **风险管理**：通过多样化投资和风险评估工具，帮助用户降低投资风险。
- **教育资源**：提供丰富的投资教育资源，帮助用户提升投资知识和技能。

## 周报
TP投资每周发布投资组合周报，详细介绍投资组合的表现、市场动态和未来展望，帮助用户了解最新的投资趋势和策略调整建议。

## SEO优化
### 已实现的SEO功能
- **优化的元标签**：每个页面都包含优化的标题、关键词和描述标签
- **robots.txt**：允许搜索引擎爬取所有页面
- **自动生成sitemap.xml**：根据data目录下的周报文件自动生成
- **内部链接优化**：使用语义化标签和优化的锚文本
- **H标签层次结构**：正确使用H1-H6标签，确保页面结构清晰
- **移动响应式设计**：确保在移动设备上有良好的用户体验
- **Google Analytics集成**：使用gtag.js跟踪网站流量

### 自动生成sitemap.xml
项目包含一个Node.js脚本，用于自动生成sitemap.xml文件：

```bash
node js/sitemap.js
```

该脚本会自动读取data目录下的所有周报JSON文件，并生成包含以下页面的sitemap.xml：
- 首页（index.html）
- 时光机页面（timeline.html）
- 所有周报详情页面（detail.html?id=xxx）

### GitHub Actions工作流
项目包含一个GitHub Actions工作流，用于自动部署到GitHub Pages：

```bash
.github/workflows/deploy.yml
```

当推送到main分支时，工作流会自动执行以下操作：
1. 检出代码
2. 设置Node.js环境
3. 生成最新的sitemap.xml
4. 部署到GitHub Pages
